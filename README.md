### "The Future of AI-Powered Identity Blending"

**Bilasian AI** is a premium web application designed for high-fidelity identity synthesis. Featuring a sophisticated **Liquid Glass** aesthetic and an Apple-inspired **Dynamic Island**, it provides a cutting-edge interface for blending subjects into cinematic compositions.

---

## ✨ Key Features

- 🎭 **Multi-Model Synthesis**: Choose between RBM Model A, Model N, and the experimental Model X.
- 📱 **Dynamic Island Interaction**: Real-time status indicators, synthesis progress, and history management all housed in a fluid, morphing UI component.
- 📊 **Genesis Core Terminal**: An integrated AI dashboard with real-time hardware telemetry (CPU/GPU/NPU/RAM) and a live log terminal.
- 🌫️ **Liquid Glass UI**: Ultra-premium glassmorphism design with frosted surfaces, suburban grit, and high-contrast "Streetwear Noir" aesthetics.
- 📂 **Synthesis History**: Persist and manage your results with a built-in gallery and "Rare" result detection.

---

## 🛠️ Technology Stack

- **Frontend**: Vanilla JavaScript (ES6+), Semantic HTML5, CSS3 (Glassmorphism, Animations)
- **Backend**: Node.js, Express.js
- **AI Core**: On-Device AI Training by Tahmid Kibria

## 🚀 Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn
- Python 3.x (for CLI tools)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Tahmid-Kibria/Bilasian-AI.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Create a `.env` file and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

### Running the App

Start the backend server:

```bash
npm start
```

The application will be available at `http://localhost:3000`.
